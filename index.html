Overview

NeuroMamba is a multimodal deep learning system that combines functional MRI brain connectivity analysis with Large Language Models for automated autism spectrum disorder diagnosis. The model processes resting-state fMRI data, learns dynamic brain connectivity patterns, and generates structured clinical reports using instruction-tuned Llama-3.

🧠

Temporal Graph Encoding

Captures dynamic brain connectivity patterns using depthwise convolutions and learned adjacency matrices over time.

🤖

LLM Integration

Projects brain embeddings into Llama-3-8B-Instruct's semantic space for natural language clinical report generation.

⚡

Efficient Training

4-bit quantization (NF4) with LoRA adapters enables training on consumer GPUs while preserving model quality.

📊

Structured Output

Generates valid JSON diagnostic reports following a predefined clinical schema for downstream integration.

Model Architecture

The system consists of three main modules: a Temporal Graph Encoder for processing fMRI signals, a Transformer-based feature refinement stage, and the frozen LLM with trainable projection layers.

Input: BOLD Signal [B, 100, 39]
100 timepoints x 39 ROIs (MSDL Atlas)

TemporalGraphEncoder
Depthwise Conv1D → BatchNorm → ReLU → Learned Adjacency
↓
Graph Convolution: h_graph = Adj @ h
Propagate features through learned connectivity
↓
TransformerEncoder (2 layers)
d_model=128, nhead=4, norm_first=True
↓
Cross-Attention Pooling
8 learnable query tokens → [B, 8, 128]
↓
Projector + LayerNorm
Linear(128, 4096) → [B, 8, 4096] (LLM hidden size)
↓
Llama-3-8B-Instruct (Frozen + LoRA)
Concatenate brain embeds with text → Generate JSON report

Key Components

1. Temporal Graph Encoder

Processes raw fMRI timeseries to extract dynamic brain connectivity patterns.

class TemporalGraphEncoder(nn.Module):
    def __init__(self, n_rois, latent_dim=64):
        super().__init__()
        self.temporal_enc = nn.Sequential(
            nn.Conv1d(n_rois, n_rois * 2, kernel_size=3, padding=1, groups=n_rois),
            nn.BatchNorm1d(n_rois * 2), nn.ReLU(),
            nn.Conv1d(n_rois * 2, n_rois, kernel_size=3, padding=1, groups=n_rois),
            nn.BatchNorm1d(n_rois), nn.ReLU()
        )
        self.time_proj = nn.Linear(MAX_SEQ_LEN, latent_dim)
        self.W_q = nn.Linear(latent_dim, latent_dim)  # Query projection
        self.W_k = nn.Linear(latent_dim, latent_dim)  # Key projection
        self.scale = latent_dim ** -0.5

    def forward(self, x):
        x = x.transpose(1, 2)  # [B, N, T]
        x_temp = self.temporal_enc(x)
        h = self.time_proj(x_temp)
        Q = self.W_q(h)
        K = self.W_k(h)
        scores = torch.matmul(Q, K.transpose(-2, -1)) * self.scale
        adj = F.softmax(scores, dim=-1)  # Learned adjacency
        return h, adj
